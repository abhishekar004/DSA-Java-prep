# 🧠 Learning Notes

### Two Sum

💡 **Concept:** HashMap for constant-time lookup

**Tip:**
Store previously visited numbers in a HashMap so we can find the complement in **O(1)** time.

**Time Complexity:** `O(n)`
**Space Complexity:** `O(n)`
